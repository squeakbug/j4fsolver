# Библиотека методов дискретного программирования

## Сборка

```sh
git clone https://github.com/squeakbug/j4fsolver
cd j4fsolver
cabal install
```

## Запуск примеров

```sh
cabal run eight-queens
```

## Тестирование

```sh
cabal test
```
